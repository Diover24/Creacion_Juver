# UberClone - Mobile Transport ApplicationA 
cross-platform mobile application inspired by Uber, built using React Native CLI for both iOS and Android. This project integrates a complete ecosystem of Google APIs to provide a seamless, real-time mobility experience , alongside a non-relational database for user authentication and data storage.  
## Key Features
### Smart Destination Search: 
Real-time suggestions using Google Places Autocomplete.  
### Interactive Maps & Routing: 
Current location rendering via Google Maps API and optimal route calculation with Google Directions API.  
### Fare & Time Estimation:
Trip distance and duration calculations using Google Distance Matrix API.
### State & Navigation Management:
Global state handling with Redux and screen transitions using React Navigation. 
### User Profile & Real-Time Tracking: 
Custom user registration with input validation and live driver tracking with animated markers.  
### Payment Gateway:
Integration with Stripe and Mercado Libre.
